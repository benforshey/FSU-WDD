An Overview of Chartist.js

  • independent JavaSript library
    • doesn't require jQuery
    • only 22KB of JavaScript and 7KB of CSS (jQuery 1.11.x weighs 96KB)
  • very customizable with SASS
  • uses SVG for illustration
  • integrates media queries for your responsive design

An Overview of Chartist

Get the Files with Bower (after bower init)


bower install chartist --save

Get the Files via GitHub

Download the files and place chartist.min.css and chartist.min.js in your appropriate folders.

HTML Setup


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Charts</title>
    <link rel="stylesheet" href="path/to/chartist.min.css">
    <script src="path/to/chartist.js"></script>
</head>
<body>
<!--    Your charts and content go here.-->
    <script src="path/to/yourJS.js"></script>
</body>
</html>

An Overview of Chartist

Start basic. In your JavaScript file: 


var data = {

  labels: ['Cars', 'Trucks', 'Planes', 'Trains', 'Boats'],
  series: [
    [6, 5, 7, 2, 0]
  ]
};

new Chartist.Line('.ct-chart', data);

That's a chart!
